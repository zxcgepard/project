Telegram Кулинарный Бот

Бот для поиска и генерации рецептов с помощью нейросети.

Функции
- Поиск рецептов по продуктам
- Генерация рецептов на ужин, десерт, случайное блюдо
- Работает 24/7 на облачном сервере

Технологии
- Python
- pyTelegramBotAPI
- Ollama (нейросеть Qwen 2.5 3B)
- Ubuntu 22.04

Установка и запуск
2. Установить зависимости: `pip install -r requirements.txt`
3. Запустить бота: `python3 bot.py`
